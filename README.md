# Job Tracker

A C# .NET desktop application built as a university project to manage job applications.

## Features
- Add, update, and track job applications
- Store data in an Oracle database
- Basic CRUD functionality

## Tech Stack
- C# (.NET)
- Oracle Database
- Visual Studio

## Running the Project
This project is intended to be opened in Visual Studio on Windows.
Database credentials are not included.

## What I Learned
- Structuring applications using services and repositories
- Working with a relational database
- Separating UI and data access logic
